Legacy Graph Coverage

Interior Series Guide

Read this as a room-by-room key for the earlier comparison charts.

Each card identifies the room or monitoring group, explains how it should be read in relation to the atlas, and notes whether it is mapped on the current plans. Raw logger IDs are kept only as source references.

Reading Notes

What the old graphs add to the current atlas.

The atlas remains the main interface, but these notes preserve what the long-range and focused graph exports helped compare across rooms, time windows, and repeated logger campaigns.

Repairs / Interventions

A chronological context layer for what the building has already lived through.

These entries are preserved as a quick-use timeline, filtered by feature category. The list supports interpretation; it does not replace technical diagnosis.
